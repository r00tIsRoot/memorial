개발동기


관광지는 국가의 중요한 재산이고 과거를 살아온 선조들이 만들어 우리들에게 남긴 고마운 선물이면서 동시에 현재를 살아가는 우리들이 자손들에게 잘 가꾸고 보존하여 남겨줘야 하는 선물이기도 하다.
국내외를 가리지 않고 여행에 대한 수요와 관심이 끊임없이 늘어나고 있는 추세이다. 하지만 늘어난 관광 수요만큼 관광객들의 시민의식은 성장하지 못했기 때문에 관광지의 낙서 등과 같은 관광지 훼손 또한 기하급수적으로 늘어났다.
후대에 남겨줘야 할 관광지의 보존을 위해 최근 이슈가 된 증강현실을 이용하여 “증강현실을 이용한 관광지 낙서장”의 개발을 시작하였다.
